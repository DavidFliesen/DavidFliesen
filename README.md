## 🤖 Key AI & Coding Projects

### 🏥 [AHA — Agentic Healthcare Assistant](https://youtu.be/QECOr-wRevM) *(Purdue Capstone · 2026)*

AI-powered healthcare assistant designed to break through the information silos limiting patient access. Features a **D-ID conversational avatar (Angel)** with multimodal voice + text interaction, role-based login (Patient / Doctor / Admin), appointment booking, patient records CRUD, RAG over clinical PDFs, and medical search. Built with LangGraph + Groq LLaMA 3.3 70B + Streamlit in Google Colab.

`LangGraph` `Groq LLaMA-3.3-70b` `Streamlit` `D-ID Avatar` `RAG` `FAISS` `Role-Based Auth` `Google Colab`
